# 🎲 D&D Session Tracker - Modular Edition

Ein vollständiger D&D 5e Session Tracker mit modularer Architektur und professionellem CR Calculator.

![Version](https://img.shields.io/badge/version-2.3.6-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![Build](https://img.shields.io/badge/build-passing-brightgreen.svg)
![Size](https://img.shields.io/badge/size-0.91MB-orange.svg)

## ✨ Features

- **⚖️ CR Calculator** - Professioneller Challenge Rating Calculator (NEU!)
- **Party Management** - Charakterverwaltung mit Spell Slots
- **Initiative Tracker** - Kampf-Reihenfolge mit HP Tracking
- **Encounter System** - Begegnungen planen und speichern
- **Quest Tracking** - Quests und Storylines
- **NPC Database** - NPCs organisieren
- **Spell Management** - Zaubersprüche tracken
- **Loot System** - Beute und Items
- **Dice Roller** - Würfel mit Statistiken

## 🚀 Quick Start

```bash
# Fertige App öffnen (keine Installation nötig!)
dist/dnd-tracker-optimized.html
```

Läuft komplett offline im Browser - kein Server, keine Installation!

## 📦 Installation

```bash
# Repository klonen
git clone https://github.com/DEIN-USERNAME/dnd-tracker-modular.git
cd dnd-tracker-modular

# App öffnen
open dist/dnd-tracker-optimized.html
```

## 🔧 Development

```bash
# Production Build
python build-optimized.py

# Dev Server
python serve.py
```
