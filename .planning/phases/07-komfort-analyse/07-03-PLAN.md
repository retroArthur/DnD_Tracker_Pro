<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| File picker → app | Untrusted local file (name, size, bytes) enters via `<input type="file">` |
| Stored metadata → DOM | File names and scene names rendered into the Soundboard tab HTML |
| Keyboard → audio engine | Alt+Shift+digit dispatches scene activation |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-AUDIO-NAME | Spoofing/XSS | file name + scene name rendered in renderAudioLibrary/renderSceneList | mitigate | `esc(file.name)` and `esc(scene.name)` on every render (Task 2); verified by `grep -c "esc("` |
| T-07-AUDIO-DOS | Denial of Service | importAudioFile | mitigate | checkAudioFileSize block-before-write at import (Task 1) reusing 07-02 guard |
| T-07-KBD-CONFLICT | Tampering (UX) | Alt+Shift+0..5 binding | mitigate | RESEARCH-verified free against the full existing shortcut set; `e.code` match avoids layout ambiguity (Assumption A4) |
| T-07-DEDUP | Tampering | new render/crud globals | mitigate | No `var X = window.X` of const-globals; clean `python build.py` |
| T-07-SC | Tampering | npm installs | accept | No package installs — native APIs only (RESEARCH § Package Legitimacy Audit) |
</threat_model>
