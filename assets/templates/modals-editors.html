📝 Markdown Syntax-Hilfe

Markdown-Shortcuts ermöglichen schnelle Formatierung in allen Editoren.

Textformatierung

**fett** → fett
*kursiv* → kursiv
~~durchgestrichen~~ → durchgestrichen
`code` → code

Import/Export

Export: Exportiere Zauber, NPCs, Quests als .md-Dateien.
→ Öffne ein Entity zum Bearbeiten und klicke "📄 Export"

Import: Importiere Markdown-Dateien zurück.
→ Öffne ein Entity und klicke "📥 Import"

Einstellungen

Markdown-Shortcuts können im Data-Tab unter "📝 Editor Features" aktiviert/deaktiviert werden.
